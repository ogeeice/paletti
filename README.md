PALETTI 🎨

A minimalist, geometric color harmony lab designed for creators, designers, and developers.

PALETTI is an interactive, browser-based color scheme generator that mathematically computes color harmonies in real-time. Featuring a high-DPI razor-sharp color wheel, dynamic geometric line overlays, and smart swatch locking, it delivers an ultra-clean, distraction-free environment to discover, fine-tune, and export beautiful color palettes.

✨ Features

Interactive Color Wheel: High-DPI (Retina) scaled color canvas ensuring smooth, pixel-perfect, and crisp color projections.

Dynamic Geometric Blueprints: Elegant, dashed vector lines trace the mathematical relationships between your active colors (forming triangles, squares, or complementary lines right on the wheel).

Harmony Rules: Instantly compute classical harmonies:

Analogous

Monochromatic

Complementary

Split Complementary

Triadic

Tetradic

Locked Swatches: Lock specific color cubes to preserve them while generating random variations for the unlocked slots.

Micro-Actions: Tapping or clicking any color swatch instantly copies the HEX value directly to your clipboard.

Smart Inputs: Double-click or focus the Base Seed input to instantly select the entire HEX code for quick editing.

Fluid & Responsive: Lovingly optimized for desktop, tablet, and mobile screens alike.

Perfect Dark/Light Contrast: Fully integrated custom-themed dark mode (#1f1f1f) and light mode (#e9eef6) backgrounds.

Royalty-Free Vector Assets: Uses only MIT/CC0-licensed SVGs, making the project completely safe for commercial use without attribution.

Instant Image Exports: Generate and download high-resolution palette design sheets as PNG.

🛠️ Tech Stack

Markup & Layout: HTML5 & Tailwind CSS

Interactions & Color Math: Vanilla ES6+ JavaScript

Canvas Rendering: HTML5 Canvas API (High-DPI optimized)

Typography: Montserrat (via Google Fonts)

💡 How to Use

Spin the Wheel: Click or drag on the color wheel to dynamically shift the base hue and saturation.

Lock Your Favorites: Hover over any of the five color cubes and click the Padlock icon to freeze that specific color.

Roll the Dice: Click the Random button to generate fresh color variations that organically drift within the color family of your current base seed.

Change Harmonies: Switch harmony modes in the dropdown menu to see different geometric patterns map across the canvas.

Double-Click HEX: Want to insert a specific starting color? Double-click or click inside the "Base Seed Color" box to instantly highlight the text and type your own code.

Export & Copy: Click any swatch to copy its HEX value, or hit the Export icon in the header to download a high-res PNG spec sheet.
